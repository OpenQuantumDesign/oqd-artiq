# OQD ARTIQ

Welcome to the OQD ARTIQ documentation.

This repository provides a Nix flake for installing specific OQD applications along with ARTIQ and all its dependencies.

## OQD Applications

- `oqd-core`
- `oqd-compiler-infrastructure`

## Installation

To install:

```bash
nix develop
```

Or to use it in a shell:

```bash
nix shell github:OpenQuantumDesign/oqd-artiq
```

## About ARTIQ

For more information about ARTIQ, visit the [official ARTIQ website](https://m-labs.hk/artiq).
